## Features

#### View types

- Current age
- Date of Birth
- Upcoming Birthday
- Remaining Birthday

#### Filter types

- Months
- Weeks
- Days

#### Behavour

- Simplicity
- Performance
- UI/UX
- Animations
- Lightweight

#### Others

- Multiple themes
- Comparing different dates of birth
